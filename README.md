# Library Management System

A simple console-based Library Management System built with Python using Object-Oriented Programming (OOP).

## Features

* Add new books
* View all books
* Borrow books
* Return books
* Track book availability
* Store books as objects inside a list

## Concepts Practiced

* Classes and Objects
* Constructors (`__init__`)
* Instance Attributes (`self`)
* Methods
* Lists of Objects
* Object State Management
* Loops and Conditions

## Example

```text
=== Library Manager ===

1. Add Book
2. Show Info Of Book
3. Borrow Book
4. Return Book
5. Quit
```

## Book Structure

Each book contains:

* Title
* Author
* Availability Status

Example:

```text
Name: Shin
Author: Jack
Available: True
```

## Technologies

* Python 3
* Object-Oriented Programming (OOP)

## Purpose

This project was created as a practice project while learning Python OOP fundamentals and working with lists of objects.
